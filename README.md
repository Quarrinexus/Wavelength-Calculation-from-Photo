This Program was used as a tool for a Cambridge University IB Physics Group Project and was designed to calculate the wavelength of a laser from a photo of a diffraction pattern, using the established model of Fraunhofer Diffraction

The software operates in the following order:
Images -> Image Filterer.py -> Filtered_Images -> Image Analysis.py -> Intensity & Lambda Calculation Graphs + Data.csv -> Data Analysis

The Image Filterer.py program may require the user to change threshold values to achieve a desirable image

The Image Analysis.py program contains and utilises default values for screen width, length from slit to screen and slit separation which can be changed according to the experiment

When going through datasets the user may be required to manually change values for min_width, min_height and data_crop_distance for the program to accurately locate the maxima
